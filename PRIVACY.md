# Privacy Policy

**LaraClaude** is an open-source Claude Code plugin that runs entirely on your local machine.

## Data Collection

This plugin does **not** collect, store, transmit, or share any user data.

## How It Works

- All skills run locally within your Claude Code session
- No external servers are contacted
- No analytics or telemetry
- No user accounts or authentication
- No cookies or tracking

## File Access

Skills may read and modify files in your project directory (migrations, models, views, etc.) only when you explicitly invoke them. All changes require your confirmation before being applied.

## Third-Party Services

This plugin does not integrate with or send data to any third-party services.

## Contact

For questions about this policy, open an issue at [github.com/edulazaro/laraclaude](https://github.com/edulazaro/laraclaude).
